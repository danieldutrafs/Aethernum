# Aethernum

Ecossistema pessoal de automação e gerenciamento. 
Desenvolvido com foco em sistemas estruturados e lógica em Java.

---

### 🌌 Visão Geral
O **Aethernum** é um projeto de longo prazo focado na integração entre software e o mundo físico. O objetivo é criar um núcleo de controle centralizado para organização de dados e autossuficiência.

### 🏗️ Estrutura Base
O sistema é organizado em dois pilares:
* **Lyra:** Assistente de comunicação e percepção. É a interface direta com o usuário.
* **Aion:** Núcleo de lógica e processamento. Responsável pela execução e automação.

---

### 🛠️ Tecnologias de Estudo
* **Linguagem:** Java
* **Persistência:** SQL
